# 前言

大家好，本次分享的毕业设计项目是基于SpringBoot的房屋租赁系统。该项目运用了Java语言，整合了多种先进的技术框架，致力于为用户提供便捷的房屋租赁体验。以下是该项目的基本介绍。

# 内容介绍

本项目是一个完整的房屋租赁系统，涵盖了房屋信息发布、租赁申请、合同签订等业务功能。系统采用前后端分离的设计模式，前端负责展示页面及交互，后端提供数据接口。通过使用Spring Boot框架，提高了项目的开发效率和稳定性。

# 技术介绍

- **语言：Java**
- **使用框架：Spring Boot**
- **前端技术：JS、Vue、css3**
- **开发工具：IDEA/Eclipse**
- **数据库：MySQL 5.7/8.0**
- **数据库管理工具：phpstudy/Navicat**
- **JDK版本：jdk1.8**
- **Maven: apache-maven 3.8.1-bin**
- **前端环境：Node.Js 12\14\16**
